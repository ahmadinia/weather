## Environment

```bash
MONGODB_URI = mongodb://db:27017/weather-it

JOB_WEATHER_FETCH = */45 * * * * *

WEATHER_API_ROUTE = http://api.openweathermap.org/data/2.5

API_KEY = YOUR API KEY

TIMEOUT_REQUEST = 3500

ENABLE_JOB = false

PORT = 3000
```
